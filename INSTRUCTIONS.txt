# AI Text Summarizer Web App

A simple and elegant web application that uses AI to summarize long texts into concise 3-4 line summaries.

## Features
- Clean, modern UI with gradient design
- Summarizes text up to 2000+ characters
- Real-time character counter
- Loading indicator during processing
- Error handling and validation
- Shows summary statistics (character reduction percentage)
- Sample text loader for testing

## Tech Stack
- **Backend**: Python FastAPI
- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **AI Library**: Sumy (LSA - Latent Semantic Analysis) + NLTK

## Prerequisites
- Python 3.8 or higher
- pip (Python package manager)

## Installation & Setup

### Step 1: Install Python Dependencies
Open a terminal/command prompt in the project directory and run:

```bash
pip install -r requirements.txt
```

This will install:
- FastAPI and Uvicorn (web framework and server)
- Sumy (text summarization library)
- NLTK (natural language processing)

### Step 2: Start the Backend Server
Run the FastAPI server:

```bash
python -m uvicorn main:app --host 127.0.0.1 --port 8000
```

You should see output like:
```
INFO:main:Summarizer initialized successfully!
INFO:     Started server process [xxxx]
INFO:     Uvicorn running on http://127.0.0.1:8000
```

The backend will be available at: http://localhost:8000

### Step 3: Open the Frontend
Simply open the `index.html` file in your web browser:
- Double-click `index.html`, OR
- Right-click → Open with → Your preferred browser

## How to Use

1. **Enter Text**: Paste or type your text into the textbox (minimum 100 characters recommended)
2. **Load Sample**: Click "Load Sample Text" to test with a pre-loaded 2000+ character example
3. **Summarize**: Click the "✨ Summarize Text" button
4. **View Results**: The summary will appear below with statistics showing the original length, summary length, and percentage reduction

## API Endpoints

### POST /summarize
Summarizes the provided text.

**Request Body:**
```json
{
  "text": "Your long text here..."
}
```

**Response:**
```json
{
  "summary": "Concise summary...",
  "original_length": 2000,
  "summary_length": 150
}
```

### GET /health
Health check endpoint to verify the API and model status.

### GET /
Root endpoint - returns API status message.

## Error Handling
The app handles various errors:
- Empty text input
- Text too short (< 100 characters)
- Backend connection issues
- Model loading failures
- Server errors during summarization

## Performance Notes
- **First Request**: Should complete in 1-3 seconds
- **Subsequent Requests**: Less than 1 second
- **Lightweight**: No large model downloads required

## Troubleshooting

### Backend won't start
- Make sure all dependencies are installed: `pip install -r requirements.txt`
- Check if port 8000 is already in use

### "Failed to connect to API" error
- Ensure the backend server is running on port 8000
- Check the terminal for any error messages
- Try accessing http://127.0.0.1:8000 in your browser (should show a JSON message)
- Make sure no firewall is blocking the connection

### Slow summarization
- First request initializes NLTK tokenizer
- For very long texts (>5000 chars), processing takes a bit longer
- The LSA algorithm is very fast compared to deep learning models

## Customization

### Change Summary Length
Edit `main.py` line 73-74:
```python
max_length = 150  # Increase for longer summaries
min_length = 50   # Decrease for shorter minimum
```

### Change Summary Algorithm
Replace the summarizer in `main.py` line 82:
```python
# For TextRank algorithm instead of LSA
from sumy.summarizers.text_rank import TextRankSummarizer
summarizer = TextRankSummarizer()
```

### Change Port
Modify the uvicorn command:
```bash
uvicorn main:app --reload --port 8080
```
Also update `index.html` line 237:
```javascript
const API_URL = 'http://localhost:8080';
```
